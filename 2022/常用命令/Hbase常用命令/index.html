HBase常用命令

发表于 2022-10-17 | 分类于 HBase
表记录条数
小表单线程统计

count “usertable”, INTERVAL => 100000

大表MapReduce统计

hbase org.apache.hadoop.hbase.mapreduce.RowCounter ‘ZHJ:DROPTEST’

scan 查询
根据rowkey范围扫描

scan ‘tablename’,{STARTROW => ‘1001’,STOPROW=>’2001’}

过滤rowkey里包含指定字符串的，这里是rowkey包含CATALOG

scan ‘SYSTEM:CATALOG’, FILTER => “RowFilter(=,’substring:CATALOG’)”

限制条数

scan ‘test’,{LIMIT => 1}

查某个字段

scan ‘test’,{COLUMNS => ‘cf:a’,LIMIT => 2}

从指定行开始取几条

scan ‘test’,{STARTROW => ‘1001’,LIMIT=>5}

检测集群监控

hbase hbck

检测hfile

hbase hfile -h -f /hfile路径

查看hbase参数

hbase org.apache.hadoop.hbase.HBaseConfiguration get

统计hfile数量
统计一张表有多少个hfile

hadoop fs -ls /hbase/data/default/tablename/*/F/* | wc -l && date +”%Y-%m-%d %H:%M:%S”

按region统计hfile

hadoop fs -du -h /apps/hbase/data/data/default/tablename/*/F/* |awk -F ‘/‘ ‘{print $8}’|sort|uniq -c|awk ‘{print $1}’|sort|uniq -c

节点退役

graceful_stop.sh node1

查看状态

hbase shell里执行

status

status ‘simple’

status ‘detailed’

创建命名空间

create_namespace ‘UD’

查看有哪些命名空间

list_namespace

删除命名空间

命名空间必须为空才能被删除

drop_namespace ‘UD’

查看有哪些表

查看某个命名空间下的表

list_namespace_tables ‘ns1’

查看所有表

list

建表语句
插入数据
禁用表

disable ‘table_name’

is_disabled ‘tablename’

启用表

enable ‘table_name’

is_enabled ‘tablename’

查看表结构信息

describe ‘table_name’

删除表

需要先禁用表

drop ‘table_name’

根据rowkey获取数据

get ‘table_name’, ‘row_key’

get ‘table_name’, ‘row_key’,’列族:列名’

表是否存在

exists ‘tablename’

删除数据

删除某个rowkey的所有数据

deleteall ‘tablename’,’rowkey’

删除某 rowkey 的某一列数据

delete ‘table_name’, ‘row_key’, ‘column_family:column’

delete ‘test’, ‘row1’,’cf:a’

清空表

truncate ‘tablename’

先disable，然后再truncate

HMaster单机启停

hbase-daemon.sh start master
hbase-daemon.sh stop master

切记 daemon后没有S，daemons是集群启停

HRegionServer单机启停

hbase-daemon.sh start regionserver
hbase-daemon.sh stop regionserver

region 均衡balancer

$ hbase shell
hbase(main):001:0> balance_switch true
hbase(main):002:0> balancer

HBase表合并

$ hbase shell

hbase(main):004:0> major_compact “tablename”

查看建表语句

describe “table_name”

查看有哪些线程（比如compact线程）

processlist

查看版本

version

查看Procedure 和 Locks 的列表

list_procedures

修改表名

disable ‘ZHJ:DROPTEST’
snapshot ‘ZHJ:DROPTEST’,’DROPTEST_snapshot’
clone_snapshot ‘DROPTEST_snapshot’, ‘ZHJ:DROPTEST_NEW’
delete_snapshot ‘DROPTEST_snapshot’
drop ‘ZHJ:DROPTEST’
