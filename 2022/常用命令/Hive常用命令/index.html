hive进程jvm参数修改

hive-env.sh

export HIVE_METASTORE_HADOOP_OPTS=”-Xmx16g”
export HADOOP_CLIENT_OPTS=”-Xmx16g”
export HIVE_SERVER2_HADOOP_OPTS=”-Xmx16g”

或者

hadoop-env.sh

export HADOOP_CLIENT_OPTS=”xxxx”

该属性优先级低于hive-env.sh

内转外

alter table TABLENAME set TBLPROPERTIES (‘EXTERNAL’=’TRUE’);

外转内

alter table TABLENAME set TBLPROPERTIES (‘EXTERNAL’=’FALSE’);

查询显示列名

set hive.cli.print.header = true;

查询显示库名

set hive.cli.print.current.db = true;

设置执行引擎为MR

set hive.execution.engine=mr;

设置mr执行队列

set mapreduce.job.queuename=root.zhj;

禁用查询统计信息

禁用统计信息，解决入库第二次报错问题

set hive.stats.autogather=false;

禁止查询的时候使用统计信息，解决select count(*) 为0问题

set hive.compute.query.using.stats=false;

表属性查看

desc formatted table_name;

Table Type显示为EXTERNAL_TABLE外部表，MANAGED_TABLE内部表

hive分区表小文件合并

alter table tablename partition(month=202203,day=20220320) concatenate;
