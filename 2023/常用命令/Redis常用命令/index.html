查看redis参数

CONFIG GET maxmemory

设置redis参数（重启失效）

CONFIG SET maxmemory 128gb

慢日志
获取慢日志配置

CONFIG GET slowlog-*

  1. slowlog-log-slower-than：指定执行时间超过多少微秒的命令请求会被记录到日志上。
  2. slowlog-max-len：指定服务器最多保存多少条慢查询日志。超出限制后，每新增一条，就将最旧的一条日志删除。

这两个参数可以在线通过CONFIG SET修改来调整

基础操作命令
命令 说明 示例
PING 测试服务器是否在线 PING → 返回 PONG
SELECT index 切换数据库（默认 0-15） SELECT 1 → 切换到数据库1
DBSIZE 查看当前数据库键总数 DBSIZE → 返回键数量
FLUSHDB 清空当前数据库 FLUSHDB
FLUSHALL 清空所有数据库（慎用！） FLUSHALL
KEYS pattern 模糊匹配键名（生产环境慎用） KEYS user:* → 匹配 user:001 等
SCAN cursor 安全遍历键（替代 KEYS） SCAN 0 MATCH user:* COUNT 100
EXISTS key 检查键是否存在 EXISTS user:001 → 1（存在）
DEL key 删除键 DEL user:001
EXPIRE key seconds 设置键的过期时间（秒） EXPIRE user:001 3600
TTL key 查看键剩余过期时间（秒） TTL user:001 → -2（键不存在）
数据结构操作命令
字符串（String）
命令 说明 示例
SET key value 设置键值 SET name "Alice"
GET key 获取键值 GET name → "Alice"
INCR key 值自增1（原子操作） INCR counter → 1
DECR key 值自减1（原子操作） DECR counter → 0
APPEND key value 追加字符串 APPEND name " Smith" → "Alice Smith"
GETRANGE key start end 获取子字符串 GETRANGE name 0 4 → "Alice"
哈希（Hash）
命令 说明 示例
HSET key field value 设置哈希字段值 HSET user:001 name "Alice"
HGET key field 获取哈希字段值 HGET user:001 name → "Alice"
HGETALL key 获取所有字段和值 HGETALL user:001
HDEL key field 删除哈希字段 HDEL user:001 age
HKEYS key 获取所有字段名 HKEYS user:001 → ["name", "age"]
列表（List）
命令 说明 示例
LPUSH key value 从左侧插入元素 LPUSH tasks "task1"
RPUSH key value 从右侧插入元素 RPUSH tasks "task2"
LPOP key 从左侧弹出元素 LPOP tasks → "task1"
LRANGE key start end 获取列表片段 LRANGE tasks 0 -1 → 所有元素
LLEN key 获取列表长度 LLEN tasks → 2
集合（Set）
命令 说明 示例
SADD key member 添加集合元素 SADD tags "redis" "db"
SMEMBERS key 获取所有集合元素 SMEMBERS tags → ["redis", "db"]
SISMEMBER key member 检查元素是否存在 SISMEMBER tags "redis" → 1
SREM key member 删除集合元素 SREM tags "db"
有序集合（Sorted Set）
命令 说明 示例
ZADD key score member 添加带分数的成员 ZADD ranking 100 "user1"
ZRANGE key start end 按分数升序获取成员 ZRANGE ranking 0 -1 WITHSCORES
ZREVRANGE key start end 按分数降序获取成员 ZREVRANGE ranking 0 10
ZSCORE key member 获取成员分数 ZSCORE ranking "user1" → 100
事务与脚本
命令 说明 示例
MULTI 开启事务 MULTI
EXEC 执行事务 EXEC
DISCARD 取消事务 DISCARD
EVAL script numkeys key [key ...] arg [arg ...] 执行 Lua 脚本 EVAL "return redis.call('GET', KEYS[1])" 1 name
持久化与备份
命令 说明 示例
SAVE 阻塞式保存 RDB 快照（生产慎用） SAVE
BGSAVE 后台异步保存 RDB 快照 BGSAVE
LASTSAVE 最后一次成功保存的时间戳 LASTSAVE → 1630000000
AOF 相关配置 通过配置文件调整 AOF 策略 appendonly yes（启用 AOF）
服务器管理
命令 说明 示例
INFO [section] 查看服务器信息（如 INFO memory） INFO memory
CONFIG GET parameter 获取配置参数 CONFIG GET maxmemory
CONFIG SET parameter value 动态修改配置参数 CONFIG SET maxmemory 2gb
CLIENT LIST 查看客户端连接列表 CLIENT LIST
SLOWLOG GET [n] 查看慢查询日志 SLOWLOG GET 10
发布订阅（Pub/Sub）
命令 说明 示例
PUBLISH channel message 向频道发布消息 PUBLISH news "Hello Redis!"
SUBSCRIBE channel 订阅频道 SUBSCRIBE news
UNSUBSCRIBE 取消订阅 UNSUBSCRIBE news
集群管理
命令 说明 示例
CLUSTER INFO 查看集群信息 CLUSTER INFO
CLUSTER NODES 列出集群节点 CLUSTER NODES
CLUSTER KEYSLOT key 计算键的哈希槽 CLUSTER KEYSLOT user:001 → 1234
